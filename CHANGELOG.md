# Changelog

All notable changes to LambdaX will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Core framework with async-first architecture
- Policy engine with YAML configuration
- Guard orchestrator with concurrency and caching
- Built-in guards: PromptInjection, Toxicity, Privacy
- FastAPI server with OpenAPI documentation
- CLI tool for inspection and server management
- Structured logging with JSON format
- Prometheus metrics endpoint
- Docker and Docker Compose support
- Comprehensive documentation
- Unit and integration tests
- Pre-commit hooks for code quality
- CI/CD pipeline with GitHub Actions

## [0.1.0] - 2024-01-XX

### Added
- Initial release
- Core guardrails framework
- Basic guard implementations
- REST API
- CLI tool
- Documentation
